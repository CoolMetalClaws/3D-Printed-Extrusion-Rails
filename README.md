# ModuRail: A Modular 3D Printed 2020 V-Slot Extrusion Rail System

https://cdn.hackclub.com/019f0d77-58dc-75f7-90a0-1484487c8981/screenshot_2026-06-24_195642.png

## Overview

This project features a Modular 2020 V-Slot Extrusion rail System along with its parts/attachments. The parts include connectors, feet, and sliding parts. All aspects of the design were created in Onshape. 

##  Features

- Snap fit connectors and smooth movement for high modularity
- Full compatibility with all 2020 V-Slot extrusion rails and their accessories
- Quick printing for small DIY projects
- Minimal material usage for quality output
- All parts designed with print tolerances for smoother movement
- STEP files provided for ONLY for slider to allow modification 
- Super Glue (Cyanoacrylate) can be used for a permanant fix between parts
- 0.2mm print tolerances allow it to be printed on almost all printers
- Ease of assembly 

DISCLAIMER: Do not use 3D printed rails for any serious/industrial applictions or in industrial machinery where failure has the potential to cause harm and losses.

## 3D Printing Settings

### Material :

 PETG or ABS recommended for structural parts that are under constant use, use PLA for prototyping and genral projects. Ensure fillament is completely dry before printing to ensure part sizing and fit, as bubbling may cause an uneven surface and a smaller/larger end product.

### Infill : 

60%+ infill recommended for structural applications, while 15-30% will work for common projects that do no require much strain. Recommended infill type is Gyroid or Tri-Hexagon infill for structural parts.

### Outer Walls : 

4-5 outer walls recommended for structural parts that are under constant use, while 3 walls work well for all other uses.

### Supports : 

None of the parts require any supports, though a brim may be required for the rails if printing vertically to ensure it doesn't fall over.
