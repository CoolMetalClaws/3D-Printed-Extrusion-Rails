# 3D-Printed-Extrusion-Rails
This project involves designing a Modular 2020 V-Slot Extrusion rail System along with its parts/attachments on Onshape. The parts Include connectors, Feet, and Sliding parts. The Hackatime projects attached were named for a seperate project involving extrusion rails, but the rails have fully been designed during the provided Hackatime hours.
